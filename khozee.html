Khozee Radio

Stream music and radio

Khozee radio was a radio and music streaming platform delivered through an andorid and web platform. Here the FM stations could stream their radio, users could save and listen the radio, music artists could premiere their song and make a premium feature availabe for full song.

View the source code

Roles and responsibilitites

  • Create responsive mobile UI following all material design guides.

  • Convert graphical mockups to mobile designs.

  • Consult with backend developers for JSON Parsing and backend systems.

  • Proper documentation of code and application.

  • Report the progress on weekly basis to the project manager.

  • Estimate and create deadlines for tasks and assign among team members through JIRA.

Challenges during the project

  • The biggest challenge was that, I was fairly new to industry and mobile development. I challenged myself to learn, practise and implement at the same time.

  • Another challenge came when I was not able to find the perfect audio player for the streaming urls of radio. Upon my research I found about the exoplayer library provided by google which saved my day.

  • Working in a fast pace environment of an office. I had to take the work to home and commit so many hours of my personal time to the implementation of the project to meet deadlines.
